❓ Perguntas Frequentes

  • Meus dados são enviados para algum servidor?
    Não. Todos os dados ficam exclusivamente no seu dispositivo.
  • Como fazer backup dos meus dados?
    Use a função "Exportar / Importar" disponível na tela inicial do app.
  • O app funciona sem internet?
    Sim, exceto para carregar os mapas de território, que requerem conexão.
  • Como trocar o idioma?
    Toque no ícone ⚙️ na tela inicial e selecione o idioma desejado.
  • Perdi meus dados após reinstalar o app.
    Os dados são armazenados localmente. Ao desinstalar, os dados são removidos. Use o backup regularmente.
